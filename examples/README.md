# Examples

This directory will contain small runnable examples for Bastion.

Planned examples:

- protected API that validates Bastion-issued access tokens;
- MCP server that advertises authorization metadata and enforces scopes;
- agent delegation demo showing user consent, scoped token issuance, resource access, and audit logs;
- minimal frontend for reviewing and revoking agent delegations.

Examples should be small, explicit, and security-conscious. They must not log bearer tokens, client secrets, private keys, or real user data.
